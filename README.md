## Тестовое задание "Web ATOM" на должность "Frontend-developer (React)".

### Реализовано:

- Каталог продуктов
- Страница одного продукта
- Добавление продукта
- Удаление продукта
- Изменение продукта

## Запустить локально

Клонировать проект

```bash
  git clone https://github.com/AlbShar/fakeStore.git
```

Перейти в директорию проекта

```bash
  cd fakeStore
```

Установить зависимости

```bash
  npm install
```

Запустить сервер разработки

```bash
  npm run start
```

### Стек:

- React
- Redux Toolkit
- Axios
- Typescript
- Tanstack Query
- React Hook Form
- Yup
- Material UI
- Styled-components
